Exam-IQ 🎓🤖

Exam-IQ is an AI-powered educational platform designed to improve digital learning by combining intelligent tools with modern web and mobile technologies.

The system allows students, lecturers, and administrators to collaborate on a unified platform where students can study smarter using AI-generated summaries and quizzes, while lecturers manage course content and evaluate learning progress.

Exam-IQ aims to simplify academic management while providing students with smart study assistance.



🚀 Features

👨‍🎓 Student Features

Students can:

Register and log into the platform

Register for courses

Access lecture materials

Receive AI-generated summaries of lecture notes

Practice with AI-generated quizzes

Take assessments online

View results and performance history

Track learning progress




👨‍🏫 Lecturer Features

Lecturers can:

Upload lecture notes and course materials

Create quizzes and assessments

Manage courses

View student participation and performance

Provide learning resources to students




🛠 Admin Features

Administrators manage the entire system including:

Managing student accounts

Managing lecturer accounts

Monitoring system activity

Managing courses

Ensuring platform integrity




🤖 AI Capabilities

Exam-IQ integrates AI to improve how students learn.

Key AI features include:

Automatic lecture note summarization

AI-generated practice quizzes

Smart study assistance

Content understanding support


These tools help students understand academic content faster and practice more effectively.

⚙️ Installation

1 Clone the Repository

git clone https://github.com/yourusername/exam-iq.git



2 Navigate to the Project Folder

cd exam-iq



3 Setup Frontend (Flutter)

cd frontend
flutter pub get
flutter run



4 Setup Backend (Next.js)

cd backend
npm install
npm run dev



5 Configure Firebase

Create a Firebase project and add your configuration inside:

firebase/firebaseConfig.js
